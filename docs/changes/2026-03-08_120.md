## APIの変更・追加

### 修正: POST /api/test-matrix/records
- 同座標・同ステータス → `{ updated: true, skipped: true }` を返してスキップ
- flaky_rate をステータス変更時に自動計算
- 次元登録済みの場合のみ座標値を検証（400 Bad Request）

### 追加: GET /api/test-matrix/:slug/gaps
```json
{
  "gaps": [
    { "coordinates": {...}, "status": "untested" },
    { "coordinates": {...}, "status": "stale_warn", "daysSince": 10 },
    { "coordinates": {...}, "status": "stale_danger", "daysSince": 35 }
  ],
  "coverageRate": 0.75,
  "totalCombinations": 8
}
```

### 追加: GET /api/test-matrix/records/:id/history
- test_historyテーブルから該当レコードの変更履歴を返す

### 修正: GET /api/test-matrix/summary
- `coverageRate`・`totalCombinations` を追加
